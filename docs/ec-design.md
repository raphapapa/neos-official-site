# EC機能 設計書

## 決定事項（2026-03-06）

### 方式
Stripe Checkout 方式（公式サイト内に商品ページ + Stripe決済）

### 要件
- 受注生産・期間限定販売
- 単価: 5,000〜150,000円
- 当初10着未満/回
- 一般価格 + NEOS価格（プロモーションコードで割引）
- 決済: クレジットカード（PayPay検討）
- 発送: ファムファム or 業務委託

### 購入フロー
1. /store でユニフォーム商品ページを表示
2. サイズ選択 → 「購入する」
3. Stripe Checkout 決済ページ（クレカ/PayPay）
4. 決済完了 → 確認メール自動送信
5. 管理側に注文通知 → 業者に発注 → 発送

### NEOS価格の実現
- Stripeプロモーションコード機能を使用
- 選手・関係者にコードを配布

### 実装フェーズ
- Phase 1: Stripeアカウント作成・商品登録・決済テスト
- Phase 2: /store ページを商品表示に変更
- Phase 3: 注文通知の仕組み（メール or Discord）
- Phase 4（将来）: 管理アプリに注文一覧を追加
