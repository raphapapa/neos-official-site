## 📄 ページ一覧

**トップ（ / ）**
サイトの顔。ヒーロー画像 → ABOUT紹介 → メンバー6名 → 最新ニュース3件 → スポンサーロゴ の構成
→ ヒーロー画像は現在未設定（グラデーション表示）

**NEWS（ /news ）**
管理アプリで公開した記事の一覧。NEWS / BLOG / RESULT のカテゴリフィルタ付き
記事クリックで詳細ページへ（Markdown本文表示）

**MEMBERS（ /members ）**
メンバー一覧。ATHLETE → GROWTH → YOUTH → JUNIOR → STAFF の順に表示
各カテゴリに英語の見出し付き。同カテゴリ内はPR昇順
カードには英語名とカテゴリバッジ。STAFFは個別役職（OWNER / OPERATOR 等）を表示
クリックで詳細ページへ（画像・PR・賞金・大会結果・Xリンク等）

---MESSAGE 3---

**ABOUT（ /about ）**
チームの理念・価値観・活動内容の紹介
MISSION → VALUES（GROWTH / TRUST / RESPECT） → ACTIVITIES → VISION の構成
管理アプリの「サイト設定 > Aboutテキスト」から内容変更可能

**SCRIM（ /scrim ）**
定期開催スクリムの紹介。「続けた先に、強さがある」をテーマにコンセプトを掲載
- APF SCRIM — NEOSとAPFの共同運営、毎週末（@APF0401）
- JUNIOR SCRIM — U13対象のソロラガ、毎週 月・水曜（@neosclan_u13）

**STORE（ /store ）**
現在「COMING SOON」表示

**PARTNERS（ /partners ）**
パートナー企業のロゴをティア別に表示。管理アプリから追加・編集可能

**CONTACT（ /contact ）**
入隊希望フォーム（名前・Epic ID・志望動機 等）と一般問い合わせフォーム
送信データは管理アプリの「入隊希望者」に自動登録

## ✅ 確認してほしいポイント

- 各ページの **テキスト・表現** に違和感はないか
- メンバーの **並び順・カテゴリ分け・英語名** は正しいか
- ABOUT の **理念・活動一覧** に過不足はないか
- SCRIM の **説明文・開催曜日** は正しいか、他に掲載すべきスクリムはあるか
- CONTACT の **入力項目** は過不足ないか
- **スマホ表示** に問題はないか（ハンバーガーメニュー含む）
- その他、追加したいページや機能はあるか

## 💬 フィードバックの書き方

以下の形式で書いてもらえると対応しやすいです
```
【ページ名】
【種別】テキスト修正 / レイアウト変更 / 画像差し替え / 追加要望 / バグ報告
【内容】具体的な修正内容や要望
```
例：
```
【ABOUT】
【種別】テキスト修正
【内容】ACTIVITIESに「VOD分析会」を追加してほしい
```
