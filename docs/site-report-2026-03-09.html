選手一覧（/members）

  • APIから全選手を取得し、5カテゴリに自動分類表示
  • 表示順：ATHLETE → GROWTH → YOUTH → JUNIOR → STAFF
  • 同カテゴリ内：PR順位が低い順（=上位選手が先）→ 名前順
  • カテゴリフィルタボタンで絞り込み可能
  • 英語名（name_en）を主表示

選手詳細（/members/[id]）

  • 画像ギャラリー（メイン画像 + サムネイルグリッド）
  • プロフィール情報（背番号、PR順位、獲得賞金、加入日）
  • 自由記述テキスト（profile, achievements）
  • 大会実績テーブル（順位3位以内は赤色ハイライト）
  • X（Twitter）アカウントリンク
  • OGP自動生成（選手名 + カテゴリ + 画像）

ニュース一覧（/news）

  • 15件/ページのページネーション
  • 3カテゴリフィルタ：NEWS / BLOG / RESULT
  • サムネイル付きカード表示

記事詳細（/news/[slug]）

  • Markdown形式の本文をHTMLにレンダリング
  • フルブリードサムネイル画像
  • X（Twitter）シェアボタン + URLコピー機能
  • 前後の記事ナビゲーション
  • OGP自動生成（記事タイトル + サムネイル画像）

お問い合わせ（/contact）

  • 2種類のフォーム切替：入隊希望（JOIN）/ 一般問い合わせ（GENERAL）
  • 入隊受付のON/OFFは管理アプリの site_settings で制御
  • 入隊フォーム：16項目（活動名、希望部門、年齢、Epic ID、X ID、デバイス等）
  • 13歳未満は保護者同意が必須
  • 送信データは管理アプリに保存
