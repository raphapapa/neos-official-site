# NEOS E-SPORTS 公式サイト レビューガイド

## サイトURL

https://neos-official-site.vercel.app

※ 検索エンジンには非公開（noindex）のため、URLを知っている人のみ閲覧可能

---

## サイトの目的

NEOS E-SPORTSの公式Webサイトとして、以下の役割を担う

- **対外的な顔** — チームの理念・活動内容・メンバーを外部に伝える
- **入隊希望者の入口** — チームに興味を持った人がコンタクトできる導線
- **スポンサー・パートナー向け** — チームの規模感・活動実績を示す
- **ファン・保護者向け** — ニュースや大会結果を閲覧できる情報ハブ

---

## データ連携の仕組み

管理アプリ（neos-player-app）で登録したデータが自動的にサイトに反映される

- メンバー情報・画像 → MEMBERS ページ
- 記事（NEWS/BLOG/RESULT） → NEWS ページ
- スポンサー情報 → トップページ・PARTNERS ページ
- サイト設定（ヒーロー画像・About文等） → トップページ・ABOUT ページ

データ更新後、最大60秒で公開サイトに反映

---

## ページ一覧と説明

### 1. トップページ（ / ）

サイトの顔となるページ
5つのセクションで構成

| セクション | 内容 |
|-----------|------|
| **HERO** | 全画面ヒーロー画像（※現在未設定、グラデーション表示）にチーム名とタグライン「GROWTH / TRUST / RESPECT」 |
| **ABOUT** | チームの理念を短く紹介し、ABOUTページへ誘導 |
| **MEMBERS** | PR上位6名のメンバーカード（画像・英語名・カテゴリバッジ） |
| **NEWS** | 最新3件の記事カード（サムネイル・タイトル・日付） |
| **SPONSORS** | パートナー企業のロゴ一覧 |

**確認ポイント:**
- ヒーロー画像を設定した際の見え方（現在は背景なし）
- ABOUT紹介文の内容
- 各セクションの表示順

---

### 2. NEWS（ /news ）

管理アプリで公開した記事の一覧ページ

- カテゴリフィルタ: ALL / NEWS / BLOG / RESULT
- 記事カード: サムネイル・タイトル・抜粋・日付・カテゴリバッジ
- ページネーション: 9件ずつ表示

記事をクリックすると詳細ページ（Markdown本文表示）へ遷移

**確認ポイント:**
- 記事カードのレイアウト
- カテゴリの分類は適切か

---

### 3. MEMBERS（ /members ）

チームメンバーの一覧ページ

- フィルタボタン: ALL / ATHLETE / GROWTH / YOUTH / JUNIOR / STAFF
- カテゴリごとにセクション分け、各セクションに見出し付き
  - ATHLETE — The frontline competitors
  - GROWTH — Rising to the next level
  - YOUTH — Future of NEOS
  - JUNIOR — Where it all begins
  - STAFF — Behind the scenes
- 表示順: ATHLETE → GROWTH → YOUTH → JUNIOR → STAFF
- 同カテゴリ内: PR昇順、PRなしは名前昇順
- カード: 画像・英語名・カテゴリバッジ（STAFFは個別役職を表示）

メンバーをクリックすると詳細ページへ遷移

**詳細ページの表示内容:**
- メイン画像 + サブ画像ギャラリー
- 英語名（大）+ 日本語名（小）
- カテゴリバッジ・背番号・PR順位・賞金・入団日
- プロフィール文・実績
- 大会結果テーブル（大会名・日付・順位・キル・賞金）
- Xアカウントリンク

**確認ポイント:**
- カテゴリの見出し・英語サブタイトルは適切か
- 並び順は意図通りか
- STAFF内の役職表記（OWNER / OPERATOR / OPERATOR SUPPORT 等）は正しいか
- 各メンバーの英語名（name_en）が設定されているか（未設定の場合は日本語名で表示）

---

### 4. ABOUT（ /about ）

チームの理念・価値観・活動内容を紹介するページ

現在の構成:
- **MISSION** — チームの使命
- **VALUES** — GROWTH / TRUST / RESPECT の3つの価値観
- **ACTIVITIES** — APFスクリム、ジュニアスクリム、1on1 等の活動一覧
- **VISION** — 卒業後に目指す姿

管理アプリの「サイト設定 > Aboutテキスト」から内容変更可能（Markdown対応）

**確認ポイント:**
- 理念・価値観の表現は適切か
- 活動一覧に過不足はないか
- 追加したい項目はあるか

---

### 5. SCRIM（ /scrim ）

NEOSが定期開催するスクリムの紹介ページ

**冒頭メッセージ:**
「続けた先に、強さがある」をテーマに、練習環境の定期開催の意義を伝える

**APF SCRIM:**
- NEOSとAPFの共同運営による朝活スクリム
- 毎週末開催
- Xアカウント: @APF0401
- バナー画像あり

**JUNIOR SCRIM:**
- U13対象のNEOS主催ソロラガ
- 毎週 月・水曜開催
- Xアカウント: @neosclan_u13
- バナー画像あり

**確認ポイント:**
- スクリムの説明文・コンセプトは適切か
- 開催曜日は正しいか
- 他に掲載すべきスクリムはあるか

---

### 6. STORE（ /store ）

オフィシャルグッズストアページ（現在「COMING SOON」表示）

**確認ポイント:**
- 将来的にどのような形でストアを運用するか（外部リンク or サイト内）

---

### 7. PARTNERS（ /partners ）

パートナー企業の紹介ページ

- ティア別にロゴを表示（GOLD / SILVER / BRONZE / STANDARD）
- 各ロゴはパートナー企業のWebサイトへリンク
- ページ下部に「パートナーシップのお問い合わせ」への導線

管理アプリの「サイト設定 > スポンサー」から追加・編集可能

**確認ポイント:**
- パートナー情報は正しいか
- ロゴ画像の表示に問題はないか

---

### 8. CONTACT（ /contact ）

お問い合わせ・入隊希望フォーム

**JOINタブ（入隊希望）:**
- 名前（必須）・メール・年齢・Epic ID・志望動機・プレイ歴・保護者名・保護者連絡先

**GENERALタブ（一般問い合わせ）:**
- 名前（必須）・メール・メッセージ

送信データは管理アプリの「入隊希望者」に自動登録される

**確認ポイント:**
- 入力項目は過不足ないか
- フォーム送信が正常に動作するか

## 共通要素

### ヘッダー（全ページ共通）
- NEOSロゴ + ナビゲーション: NEWS / MEMBERS / ABOUT / SCRIM / STORE / PARTNERS / CONTACT
- スクロールで背景が暗くなる
- スマホではハンバーガーメニュー

### フッター（全ページ共通）
- NEOSロゴ + タグライン「GROWTH / TRUST / RESPECT」
- ナビゲーションリンク
- Xアカウント: NEOS（メイン）+ JUNIOR
- コピーライト

---

## フィードバックの出し方

以下の形式でまとめていただけるとスムーズに対応できます

```
【ページ名】
【種別】テキスト修正 / レイアウト変更 / 画像差し替え / 追加要望 / バグ報告
【内容】具体的な修正内容や要望
```

例:
```
【ABOUT】
【種別】テキスト修正
【内容】ACTIVITIESに「VOD分析会」を追加してほしい
```
